# 📅 6-Месячный План Обучения Full-Stack Разработке (24 недели)

**Общий подход:**  
✅ 60–70 % времени — кодинг и мини-проекты  
✅ 30–40 % — теория и видео  
✅ 12–18 часов в неделю (реалистично для работающего/студента)  
✅ Только **бесплатные** ресурсы  
✅ Основной стек: HTML → CSS → JS → Git → React + Vite + Tailwind → Node.js + Express → MongoDB → REST API → Deploy  
✅ Дополнения 2025–2026: Tailwind, TypeScript (с месяца 5), Next.js (опционально в конце)  
✅ После каждой 4-й недели — **Контрольная неделя** (повтор + крупный проект + деплой)

---

## Месяц 1: Уверенное владение основами фронтенда (HTML + CSS + базовый JS)
**Цель месяца:** Верстать современные landing-страницы без подсказок.

### Неделя 1: HTML5 + семантика
**Темы:** теги, формы, доступность, SEO-базис  
**Ресурсы:**  
- freeCodeCamp «Responsive Web Design» (первые 3 модуля) → https://www.freecodecamp.org/learn/responsive-web-design/  
- MDN HTML basics → https://developer.mozilla.org/ru/docs/Learn/HTML/Introduction_to_HTML  
**Время:** 12–14 ч  
**Задания:**  
1. Сверстать личную карточку (визитку)  
2. Форма обратной связи с валидацией через атрибуты  
**Закрепление:** YouTube — Web Dev Simplified «HTML Full Course» (первые 2 ч)

### Неделя 2: CSS + Flexbox + Grid
**Темы:** селекторы, box-model, Flexbox, CSS Grid, responsive  
**Ресурсы:**  
- freeCodeCamp (модули 4–7)  
- The Odin Project «Foundations» → https://www.theodinproject.com/paths/foundations  
- MDN Flexbox/Grid  
**Время:** 14 ч  
**Задания:**  
1. Клон Google-страницы (только CSS)  
2. Responsive dashboard с Grid  
**Закрепление:** Net Ninja «CSS Flexbox & Grid Playlist» (YouTube)

### Неделя 3: JavaScript основы
**Темы:** переменные, функции, массивы, объекты, циклы, DOM  
**Ресурсы:**  
- freeCodeCamp «JavaScript Algorithms and Data Structures» (первые 6 модулей) → https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures-v8/  
- JavaScript.info (главы 1–5) → https://javascript.info/  
**Время:** 15 ч  
**Задания:**  
1. Интерактивный калькулятор  
2. To-Do список на чистом JS (add/delete)  
**Закрепление:** Traversy Media «JavaScript Crash Course 2025»

### Неделя 4 (Контрольная): Повтор + первый проект
**Задания:**  
- Сделать полноценный landing-page (3 секции + форма)  
- Загрузить на GitHub Pages  
- Повторить все пробелы (MDN + freeCodeCamp quizzes)  
**Результат:** 1 готовый проект в портфолио ✅
